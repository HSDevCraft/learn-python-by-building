# Changelog

All notable changes to the Python Course will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-04-27

### Added
- Initial release of complete Python course
- 15 comprehensive modules covering fundamentals to advanced topics
- 5 learning paths for different roles (Software Developer, Data Scientist, AI/LLM Engineer, Full Course, Quick Start)
- PROJECTS.md with 11 hands-on projects (5 mini, 3 intermediate, 3 capstone)
- Each module includes:
  - Learning objectives
  - Conceptual foundations with examples
  - Annotated code snippets
  - Best practices tables
  - Exercises with full solutions
  - Module summaries
  - 10-question quizzes with answers
- GitHub-ready structure
- Contributing guidelines
- MIT License

### Modules
1. Python Fundamentals — variables, types, operators, control flow
2. Functions and Scope — LEGB rule, closures, decorators, lambda
3. Data Structures — lists, tuples, dicts, sets, comprehensions
4. Object-Oriented Programming — classes, inheritance, SOLID, dataclasses
5. File Handling & Exceptions — context managers, pathlib, custom exceptions
6. Standard Library — os, sys, datetime, itertools, re, typing
7. Virtual Environments & Packages — venv, pyproject.toml, Poetry, PyPI
8. Testing — pytest, fixtures, mocking, TDD, coverage
9. Debugging & Performance — pdb, logging, profiling, asyncio, GIL
10. APIs & Web — requests, REST, FastAPI, Pydantic, httpx
11. Data Analysis — NumPy, Pandas, vectorisation, broadcasting
12. Scripting & Automation — argparse, click, BeautifulSoup, openpyxl
13. Version Control & Git — Git model, workflows, branching, GitHub Actions
14. Production Code — project structure, mypy, Black, Ruff, Docker
15. Algorithms & DSA — Big O, data structures, sorting, DP, graphs
